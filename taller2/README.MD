# 📌 Descripción del Taller

Este taller tiene como objetivo aplicar conceptos fundamentales de análisis de datos utilizando la librería **Pandas** en Python, desarrollando operaciones matemáticas, estadísticas descriptivas y procesos de manipulación de datos.

Durante el desarrollo del taller se trabajará con un conjunto de datos relacionado con productos tecnológicos y de oficina, permitiendo realizar cálculos de ventas, porcentajes, análisis estadísticos y filtrado de información.

El desarrollo completo del proyecto será realizado en **Google Colab**, permitiendo una ejecución práctica, organizada y documentada del código.

---

# 🎯 Objetivos

## Objetivo General

Aplicar técnicas básicas de análisis de datos utilizando la librería Pandas en Python.

## Objetivos Específicos

* Crear y manipular DataFrames.
* Realizar operaciones matemáticas entre columnas.
* Aplicar análisis estadístico descriptivo.
* Filtrar y organizar información.
* Generar nuevas columnas calculadas.
* Interpretar resultados obtenidos mediante Pandas.
* Exportar información procesada a archivos Excel.
